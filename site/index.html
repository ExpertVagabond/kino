Architecture
Seven crates. Use what you need.

Kino is a workspace of independent Rust crates. Import the whole stack or cherry-pick individual modules.

Application Layer
kino-desktop GStreamer native player
kino-tauri Cross-platform desktop
kino-wasm Browser via WebAssembly
kino-cli Terminal & pipelines
Core
kino-core HLS/DASH, ABR, buffers, DRM
Extensions
kino-frequency Audio fingerprinting & analysis
kino-python PyO3 bindings for Python
Capabilities
Everything a video engine needs

Production-grade features built into the core. No plugins, no third-party runtime.

⚡
Hardware Acceleration

VA-API, VideoToolbox, NVDEC, D3D11VA. Automatic backend detection. Falls back to software gracefully.

📡
Adaptive Streaming

HLS and DASH with hybrid ABR (BOLA + throughput). Buffer-aware quality switching. Zero-stall playback.

🎵
Audio Fingerprinting

Shazam-style spectral peak matching. Content identification, duplicate detection, and similarity scoring.

🧩
WebAssembly

Full player in WASM. Drop a script tag, get adaptive streaming. No native install. Works on any modern browser.

🔐
DRM Support

Widevine, FairPlay, PlayReady integration points. Content protection for commercial deployments.

✅
Formally Verified

TLA+ specifications for critical paths. 25+ invariants including NoDoubleCharge and ExactlyOnceDelivery.

use kino_core::{PlayerConfig, PlayerSession};
use kino_desktop::{DesktopPlayer, DesktopPlayerConfig};

fn main() -> anyhow::Result<()> {
    // Hardware acceleration is automatic
    let mut player = DesktopPlayer::new(
        DesktopPlayerConfig::default()
    )?;

    // Load HLS stream
    player.load("https://stream.example.com/live.m3u8")?;
    player.play();

    // Quality metrics are always available
    let metrics = player.quality_metrics();
    println!("Resolution: {:?}", metrics.resolution);

    Ok(())
}
from kino_frequency import FrequencyAnalyzer, Fingerprinter

# Analyze audio content
analyzer = FrequencyAnalyzer(fft_size=2048)
result = analyzer.analyze_file("content.wav")

print(f"Dominant frequency: {result.dominant_hz} Hz")
print(f"Genre prediction:   {result.tags.genre}")

# Generate audio fingerprint
fp = Fingerprinter()
fingerprint = fp.fingerprint("content.wav")
print(f"Hash: {fingerprint.hash}")

# Compare two files for similarity
score = fp.compare("file_a.wav", "file_b.wav")
print(f"Similarity: {score:.1%}")
